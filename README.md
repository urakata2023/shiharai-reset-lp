# fudousan ｜ 支払いリセット計画 LP

借金・ローン返済に苦しむ正社員向けの感情訴求型ダイレクトレスポンスLP。
ブランド名は **【支払いリセット計画】**。テーマは「借金の悩み」ではなく **「毎月の支払いから解放されたい」**。
住宅販売ではなく、**住まい×家計再建×ライフプラン改善** の無料相談への申込を最大化する。

## 狙う見込み客（重要）
**「信用ブラックではない・正社員でちゃんと働いている・でも借金の返済が毎月苦しい」層**を最大限集める。
この層は「まだ使える信用」が残っており、住まいと一緒に低金利で借入を組み替えられる可能性がある＝商材の核。
- FV直後の「30秒セルフチェック」1問目に信用情報（債務整理・長期滞納なし）を置き、対象者の選別＆自分ごと化。
- フォームに「過去の返済状況」設問を追加し、事故情報なしのリードを優先できるように。

## オファー / CTA
**「借金を0にできるか、無料で相談する」** に統一（断定保証はせず"相談ベース"）。
ロジック：信用がキレイ → その信用で高金利の借金を住まいと低金利で組み替え → 借金が資産に変わる。

- `assets/` … 画像素材
  - **使用中（Genspark生成のリアル写真・Web用JPEGに圧縮済 計約1.4MB）**:
    hero.jpg / advisor.jpg / case1_keiba.jpg / case2_ribo.jpg / case3_couple.jpg / case4_pachinko.jpg / case5_family.jpg / ogp.jpg
  - `assets/genspark/` … 生成オリジナル（高解像度バックアップ）
  - `assets/logo.png` … ロゴ（白背景を透過処理＋余白トリミング済）。ヘッダーはそのまま、フッターはCSS `filter:brightness(0) invert(1)` で白抜き表示。
  - `assets/favicon.png` … ロゴのマーク部分から生成したファビコン（タブ/ホーム画面用）
  - ソコスト素材（family4等のイラスト一式）は現在未使用。差し替えたい場合の予備として残置。出典: https://soco-st.com/

## デザイン
- 配色：温かいクリーム地 × 深いエバーグリーン（安心・再生） × 琥珀アクセント（希望・CTA統一）

## フォーム & 計測（実装済み・要ID設定）
- リード保存：**Googleスプレッドシート（Apps Script）**。`gas-code.gs` を設置 → ウェブアプリURLを `index.html` の `var GAS_ENDPOINT = ''` に貼る（手順は gas-code.gs 冒頭）。
- CV計測：申込完了で **Google広告（gtag conversion）/ GA4（generate_lead）/ Meta（fbq Lead）** を発火。`index.html` の `<head>` 内 `XXXX` を実IDに差し替え：
  - GA4測定ID `G-XXXXXXXXXX` ／ Google広告ID `AW-XXXXXXXXXX` ／ Google広告CVラベル（JS `fireConversions` 内 `AW-.../...`）／ Meta Pixel `XXXXXXXXXXXXXXX`
- ID未設定でもLPは動作（フォームはデモ表示、タグは空振り）。
